The SRI Labs PR Intelligence Dashboard is for internal team use. The UI should emulate professional brand watching companies — clean, enterprise-grade, data-dense but not cluttered.

**Why:** User wants the team to have a polished, trustworthy-looking tool comparable to paid brand monitoring services.

**How to apply:** Design decisions should favor clean data presentation, professional typography, dashboard-grade layouts with sidebar nav, KPI cards with trends, and clean mention tables. Avoid toy/hobby-project aesthetics.
